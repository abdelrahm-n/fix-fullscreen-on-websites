# Simple Fullscreen fix on various websites
Code made by: [@graphpg](https://discord.com/users/956614890766876784)

If your fails to open a movie, video or anything with fullscreen, copy the paste it in your browser Console:

1. Open the tab with the movie, game, video, ...
2. Press Control + Shift + I
3. Click on controls (on the top menu section)
4. Paste the code and Enter
5. Enjoy


! If the pasting doesn't work in the console, type "allow pasting" in the console, and then repaste the code
